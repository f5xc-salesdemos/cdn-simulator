Cuando la sesión de laboratorio esté completa, destruya todos los recursos de Azure para detener la facturación.

## Terraform Destroy

Desde el directorio que contiene la configuración de Terraform:

```bash
# Review what will be destroyed
terraform plan -destroy

# Destroy all resources
terraform destroy
```

Terraform solicitará confirmación. Escriba `yes` para continuar.

## Verificar la limpieza

Confirme que todos los recursos han sido eliminados:

```bash
# Check resource group is gone
az group show --name "$(terraform output -raw resource_group_name)" 2>&1 \
  | grep -q "ResourceGroupNotFound" \
  && echo "Resource group deleted" \
  || echo "Resource group still exists"
```

## Limpiar el estado local

Elimine el estado de Terraform y los archivos en caché:

```bash
rm -rf .terraform .terraform.lock.hcl terraform.tfstate terraform.tfstate.backup
```

## Revertir cambios de DNS/Hosts

Si modificó los archivos DNS o hosts durante la [integración](06-integrate/), revierta esos cambios:

```bash
# Remove hosts file entry (Linux/macOS)
sudo sed -i '/<CDN_EDGE_PUBLIC_IP>/d' /etc/hosts
```

Si creó un registro DNS A, elimínelo a través de su proveedor de DNS.
