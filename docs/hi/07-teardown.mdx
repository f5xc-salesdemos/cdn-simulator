जब लैब सत्र पूरा हो जाए, तो बिलिंग रोकने के लिए सभी Azure संसाधनों को नष्ट करें।

## Terraform Destroy

Terraform कॉन्फ़िगरेशन वाली डायरेक्टरी से:

```bash
# Review what will be destroyed
terraform plan -destroy

# Destroy all resources
terraform destroy
```

Terraform पुष्टि के लिए संकेत देगा। आगे बढ़ने के लिए `yes` टाइप करें।

## क्लीनअप सत्यापित करें

पुष्टि करें कि सभी संसाधन हटा दिए गए हैं:

```bash
# Check resource group is gone
az group show --name "$(terraform output -raw resource_group_name)" 2>&1 \
  | grep -q "ResourceGroupNotFound" \
  && echo "Resource group deleted" \
  || echo "Resource group still exists"
```

## स्थानीय State साफ़ करें

Terraform state और कैश्ड फ़ाइलें हटाएँ:

```bash
rm -rf .terraform .terraform.lock.hcl terraform.tfstate terraform.tfstate.backup
```

## DNS/Hosts परिवर्तन पूर्ववत करें

यदि आपने [इंटीग्रेशन](06-integrate/) के दौरान DNS या hosts फ़ाइलें संशोधित की थीं, तो उन परिवर्तनों को पूर्ववत करें:

```bash
# Remove hosts file entry (Linux/macOS)
sudo sed -i '/<CDN_EDGE_PUBLIC_IP>/d' /etc/hosts
```

यदि आपने DNS A रिकॉर्ड बनाया था, तो उसे अपने DNS प्रोवाइडर के माध्यम से हटाएँ।
