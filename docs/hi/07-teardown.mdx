जब लैब सत्र पूरा हो जाए, तो बिलिंग रोकने के लिए सभी Azure संसाधनों को नष्ट कर दें।

## Terraform Destroy

Terraform कॉन्फ़िगरेशन वाली डायरेक्टरी से:

```bash
# समीक्षा करें कि क्या नष्ट किया जाएगा
terraform plan -destroy

# सभी संसाधनों को नष्ट करें
terraform destroy
```

Terraform पुष्टि के लिए पूछेगा। आगे बढ़ने के लिए `yes` टाइप करें।

## सफ़ाई की पुष्टि करें

पुष्टि करें कि सभी संसाधन हटा दिए गए हैं:

```bash
# जांचें कि रिसोर्स ग्रुप हटा दिया गया है
az group show --name "$(terraform output -raw resource_group_name)" 2>&1 \
  | grep -q "ResourceGroupNotFound" \
  && echo "Resource group deleted" \
  || echo "Resource group still exists"
```

## स्थानीय State की सफ़ाई

Terraform state और कैश्ड फ़ाइलें हटाएं:

```bash
rm -rf .terraform .terraform.lock.hcl terraform.tfstate terraform.tfstate.backup
```

## DNS/Hosts परिवर्तन पूर्ववत करें

यदि आपने [एकीकरण](06-integrate/) के दौरान DNS या hosts फ़ाइलों में बदलाव किए थे, तो उन परिवर्तनों को पूर्ववत करें:

```bash
# hosts फ़ाइल प्रविष्टि हटाएं (Linux/macOS)
sudo sed -i '/<CDN_EDGE_PUBLIC_IP>/d' /etc/hosts
```

यदि आपने DNS A रिकॉर्ड बनाया था, तो उसे अपने DNS प्रदाता के माध्यम से हटा दें।
