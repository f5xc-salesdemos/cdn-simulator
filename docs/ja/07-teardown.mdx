ラボセッションが完了したら、課金を停止するためにすべての Azure リソースを破棄します。

## Terraform Destroy

Terraform 構成ファイルが含まれるディレクトリから実行します：

```bash
# 破棄される内容を確認
terraform plan -destroy

# すべてのリソースを破棄
terraform destroy
```

Terraform が確認を求めます。続行するには `yes` と入力してください。

## クリーンアップの確認

すべてのリソースが削除されたことを確認します：

```bash
# リソースグループが削除されたことを確認
az group show --name "$(terraform output -raw resource_group_name)" 2>&1 \
  | grep -q "ResourceGroupNotFound" \
  && echo "Resource group deleted" \
  || echo "Resource group still exists"
```

## ローカルステートのクリーンアップ

Terraform のステートファイルとキャッシュファイルを削除します：

```bash
rm -rf .terraform .terraform.lock.hcl terraform.tfstate terraform.tfstate.backup
```

## DNS/ホストファイルの変更を元に戻す

[統合](06-integrate/)の手順で DNS やホストファイルを変更した場合は、それらの変更を元に戻してください：

```bash
# ホストファイルのエントリを削除（Linux/macOS）
sudo sed -i '/<CDN_EDGE_PUBLIC_IP>/d' /etc/hosts
```

DNS A レコードを作成した場合は、DNS プロバイダーを通じて削除してください。
