Quando a sessão de laboratório estiver concluída, destrua todos os recursos do Azure para interromper a cobrança.

## Destruição com Terraform

No diretório que contém a configuração do Terraform:

```bash
# Review what will be destroyed
terraform plan -destroy

# Destroy all resources
terraform destroy
```

O Terraform solicitará confirmação. Digite `yes` para prosseguir.

## Verificar Limpeza

Confirme que todos os recursos foram removidos:

```bash
# Check resource group is gone
az group show --name "$(terraform output -raw resource_group_name)" 2>&1 \
  | grep -q "ResourceGroupNotFound" \
  && echo "Resource group deleted" \
  || echo "Resource group still exists"
```

## Limpar Estado Local

Remova o estado do Terraform e os arquivos em cache:

```bash
rm -rf .terraform .terraform.lock.hcl terraform.tfstate terraform.tfstate.backup
```

## Reverter Alterações de DNS/Hosts

Se você modificou o DNS ou os arquivos hosts durante a [integração](06-integrate/), reverta essas alterações:

```bash
# Remove hosts file entry (Linux/macOS)
sudo sed -i '/<CDN_EDGE_PUBLIC_IP>/d' /etc/hosts
```

Se você criou um registro DNS do tipo A, exclua-o por meio do seu provedor de DNS.
