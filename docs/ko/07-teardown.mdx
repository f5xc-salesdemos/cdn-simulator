실습 세션이 완료되면 요금 부과를 중단하기 위해 모든 Azure 리소스를 삭제합니다.

## Terraform Destroy

Terraform 구성 파일이 있는 디렉터리에서 실행합니다:

```bash
# 삭제될 항목 확인
terraform plan -destroy

# 모든 리소스 삭제
terraform destroy
```

Terraform이 확인을 요청합니다. 계속 진행하려면 `yes`를 입력합니다.

## 정리 확인

모든 리소스가 제거되었는지 확인합니다:

```bash
# 리소스 그룹이 삭제되었는지 확인
az group show --name "$(terraform output -raw resource_group_name)" 2>&1 \
  | grep -q "ResourceGroupNotFound" \
  && echo "Resource group deleted" \
  || echo "Resource group still exists"
```

## 로컬 상태 정리

Terraform 상태 및 캐시 파일을 제거합니다:

```bash
rm -rf .terraform .terraform.lock.hcl terraform.tfstate terraform.tfstate.backup
```

## DNS/Hosts 변경 사항 되돌리기

[통합](06-integrate/) 과정에서 DNS 또는 hosts 파일을 수정한 경우 해당 변경 사항을 되돌립니다:

```bash
# hosts 파일 항목 제거 (Linux/macOS)
sudo sed -i '/<CDN_EDGE_PUBLIC_IP>/d' /etc/hosts
```

DNS A 레코드를 생성한 경우 DNS 제공업체를 통해 삭제합니다.
