เมื่อเซสชันแล็บเสร็จสิ้น ให้ทำลายทรัพยากร Azure ทั้งหมดเพื่อหยุดการเรียกเก็บค่าใช้จ่าย

## การทำลายด้วย Terraform

จากไดเรกทอรีที่มีการกำหนดค่า Terraform:

```bash
# ตรวจสอบสิ่งที่จะถูกทำลาย
terraform plan -destroy

# ทำลายทรัพยากรทั้งหมด
terraform destroy
```

Terraform จะถามเพื่อยืนยัน พิมพ์ `yes` เพื่อดำเนินการต่อ

## ตรวจสอบการล้างข้อมูล

ยืนยันว่าทรัพยากรทั้งหมดถูกลบออกแล้ว:

```bash
# ตรวจสอบว่า resource group ถูกลบแล้ว
az group show --name "$(terraform output -raw resource_group_name)" 2>&1 \
  | grep -q "ResourceGroupNotFound" \
  && echo "Resource group deleted" \
  || echo "Resource group still exists"
```

## ล้างสถานะในเครื่อง

ลบสถานะ Terraform และไฟล์แคช:

```bash
rm -rf .terraform .terraform.lock.hcl terraform.tfstate terraform.tfstate.backup
```

## ย้อนกลับการเปลี่ยนแปลง DNS/Hosts

หากคุณแก้ไข DNS หรือไฟล์ hosts ระหว่าง[การเชื่อมต่อ](06-integrate/) ให้ย้อนกลับการเปลี่ยนแปลงเหล่านั้น:

```bash
# ลบรายการในไฟล์ hosts (Linux/macOS)
sudo sed -i '/<CDN_EDGE_PUBLIC_IP>/d' /etc/hosts
```

หากคุณสร้างระเบียน DNS A ให้ลบผ่านผู้ให้บริการ DNS ของคุณ
